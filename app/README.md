# app/ (Evolving Project)

This folder is the **single codebase that evolves across the 30 days**.

- Day 1 starts with a simple local-LLM client + CLI.
- Later days extend this same app into a chatbot, RAG, tools, LangGraph, and multi-agent workflows.

Note:
- Each `dayX/app1/` folder is an **independent demo mini-project** (separate from this evolving codebase).
