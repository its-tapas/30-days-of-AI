# Day 1 Independent Demo (app1)

This folder is a **separate, independent mini-project** for Day 1.

Rules:
- It should NOT import code from the root `app/` project.
- Keep it minimal and focused on the day’s concept.

Why this exists:
- `app/` is the evolving real system.
- `day1/app1/` is a small standalone demo so you can practice “building from scratch” without affecting the main codebase.
